# 🤖 AI Resume Generator

A conversational AI-powered resume generator built using Streamlit, LangChain, and Groq.

## 🚀 Features

- Step-by-step form-based interaction
- AI-generated professional resume
- Structured JSON output
- Clean UI using Streamlit
- Download resume as JSON
- Modular code architecture

## 🧠 Tech Stack

- Python
- Streamlit
- LangChain
- Groq API
- dotenv
